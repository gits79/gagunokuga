# 2025.02.05
### 오늘 한 일 
- 벡엔드에 전체 프로젝트 배포포

### 깨달은 점
- 간단한 생각으로 vue프로젝트는 5173포트로 띄우고 벡엔드는 8080으로 띄워서 접근하면 되는줄 알았음
- 하지만 SSAFY의 EC2는 5173포트로 접근 불가 특정 원하는 포트도 열 수 없음 그래서 Nginx를 사용해서 벡엔드 프론트엔드 모두 80 접근함 
- 그랬더니 이게 프론트로 접근하는건지 벡엔드로 접근하는 건지 구분이 안됨 찾아보니 벡엔드의 경우 API 경로를 시작할 때 /api 를 붙여서 구분함, 팀원들에게 전파해서 적용 완료료
### 내일 할 일
- CI/CD 파이프라인 구축축
